CSS + HTML Soulbot Orb

This orb is built with pure HTML, CSS, and a tiny bit of vanilla JS for state switching. You can wire it to your chatbot states: idle, listening, thinking, speaking, and writing.

Writing state: shows a cloudy thought bubble with animated dots. Use it when the assistant is generating text, before the bot message appears in chat.
Example integration: setSoulbotState('listening') when mic is active setSoulbotState('thinking') after user input is received setSoulbotState('speaking') while TTS audio is playing setSoulbotState('writing') while the answer is streaming into chat setSoulbotState('idle') when the bot finishes
